**Come and work for a leading North East technology consultancy, which prides itself on empowering its people, providing real value to our clients by delivering sustainable software solutions with long lasting benefits. We are seeking a dynamic and forward-thinking Automation Test Analyst, who can help us transform our major clients. You must be passionate about technology and about testing IT systems, utilising XP and agile practices. You must have an open mind and be willing to learn new technologies.


### Skills, qualifications and experience required: 

* Experience testing APIs and building automated API test suites, using REST Assured or Karate
* Knowledge of performance testing tools and frameworks, such as JMeter and Gatling
* An understanding of testing security and using security testing tools like OWASP ZAP
* Familiar with Git and have a good understanding of continuous integration tools like Jenkins, GitlabCI and CircleCI. 
* Experience with Selenium Webdriver and using/building Cucumber frameworks. 
* QA’s will be required to perform accessibility testing using tools such as JAWS and WAVE.
* Good communication and client relationship skills are essential.
